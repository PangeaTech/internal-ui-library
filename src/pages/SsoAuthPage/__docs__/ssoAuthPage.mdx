# Microsoft SSO Login

This component enables Single Sign-On (SSO) using Microsoft login for users with a Microsoft ID.

## Props

- **buttonLabel**: Label for the login button.
- **logoUrl**: URL of the logo image to display in the component.
- **onLogin**: Callback function to handle the login process.
