# Username Password Login

This component facilitates traditional username/password authentication, user signup, and initiating the password reset process.

## Props

- **mode**: Determines the form mode, can be "login", "signup", or "forgotPassword".
- **fields**: Array of objects representing the form fields.
- **logoUrl**: URL of the logo image to display in the component.
- **onSubmit**: Callback function to handle the login process.
