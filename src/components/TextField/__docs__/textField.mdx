# TextField

TextField component with different props.

#### Example

<Canvas of={TextField.Primary} />

## Usage

```ts
import {TextField} from "my-package-name";

const Example = () => {
  return (
      <TextField
        placeholder={"Type here"}
      />
  );
};

export default Example;
```

#### Arguments

- **placeholder** _`() => void`_ - A string that represents the placeholder content of the text field.
